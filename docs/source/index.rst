日麻 -- 立直麻将
===================================

介绍
----------

日麻是一个和牌得 **分数** 的排名游戏，通常每人25000分的筹码。

分数主要是由 :ref:`番` 和 :ref:`符` 决定。番的影响大，符的影响小，番太大了就不看符了。

有 :ref:`役` 才能胡。

通常使用34种共136张牌（没有 春夏秋冬、梅兰竹菊）。

可以吃。

舍牌（打出的牌）要从左往右，从远到近，每行6个排排好。

荣和 -- 和出铳在日麻叫荣和。

最后7墩牌叫王牌（ace）不摸。最后2墩（4张）叫岭上牌，杠的时候补牌摸。

特殊
----------

立直 -- 一种一番的役
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

听牌时（必须门前清）拿1000点筹码放在列牌区前，表示“立直”，其后不能改牌、吃、碰（但在不改变听牌牌型，即如有四张牌全部不在顺子中，可以做暗杠）。立直和牌时，上方表宝牌开出多少张，下方就开多少张里悬赏（杠里悬赏同）。


振听 -- 有些状况下不能荣和只能自摸
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

日麻特有规则。一旦进入振听状态，将会无法和由他人打出的牌， **只能自摸** 。

有舍牌振听、同巡振听、立直振听3种。

* 舍牌振听：自己所听的牌包含自己曾经打出的牌，包括被叫走的牌（吃碰杠）。必须透过改变手牌， **使所听的牌不再包含自己曾经打出的牌** 才能解除。当然如果自己已经宣告立直，因为立直后无法改变手牌，所以当局结束前无法解除。

* 同巡振听：他人打出自己所听的牌，但故意（要么忘了？） **不和** 造成，直到下一次自己出牌后自动解除。

* 立直振听：自己宣告立直后，自己摸到或他人打出自己所听的牌，但因为某些原因不和牌而造成，该振听于当局结束前无法解除。 


悬赏 -- 和牌的时候有指定的牌（宝牌，也叫Dora）
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

宝牌可分为赤宝牌、表宝牌、里宝牌、杠宝牌、杠里宝牌共5种，其中除赤宝牌外其余4种皆须透过 **悬赏指示牌** 获得。

和牌时，若手上有该张牌， **每张多算一番** 。

赤宝牌：红色的五万、五饼、五条各一张共3张


表宝牌：即配牌后于王牌第三墩上层所翻开之牌的 **下一张牌** 。

    下一张：

	万条饼：1→2→3→......→9→1→2......

	风牌：东→南→西→北

	番牌（三元牌）：白→发→中

杠宝牌：每开杠一次，就于表宝牌逆时方向（往回）隔壁再开一张宝牌指示牌，因日麻将只备用4张岭上牌，所以最多4杠，开4张（加上开局的指示牌则最多为5张）。

里宝牌指示牌、杠里宝牌指示牌：即为表悬赏指示牌与杠悬赏指示牌下层之牌，只有 **“立直和牌者”** 才可以开此指示牌。
